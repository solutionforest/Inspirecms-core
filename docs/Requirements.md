# Requirements

To use InspireCMS Core, ensure your environment meets the following requirements:

- PHP 8.0 or higher
- Composer
- [Laravel 11](https://laravel.com/docs/11.x)
- MySQL 5.7+ or PostgreSQL 10+
- Node.js and npm (for asset compilation)
- Filament >= [3.0](https://filamentphp.com/docs/3.x)
- A web server such as Apache or Nginx
