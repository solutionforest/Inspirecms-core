# Core Concepts

InspireCMS Core is built around the following key concepts:

## Resources

Resources represent content types in your CMS. They define fields, validation rules, and admin interfaces.

## Plugins

Plugins extend the functionality of InspireCMS. They can add features, modify behavior, or integrate with third-party services.

## Configuration

The `config/inspirecms.php` file allows you to customize the CMS, including routes, middleware, branding, and caching.
