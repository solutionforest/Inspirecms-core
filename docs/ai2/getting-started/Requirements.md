# Requirements

Before installing InspireCMS, ensure your environment meets the following requirements:

## Server Requirements

- PHP 8.3 or higher
- Laravel 11.x
- MySQL 5.7+ or PostgreSQL 10+
- Composer 2.0+
- Node.js 18+ and npm (for asset compilation)

## PHP Extensions

The following PHP extensions must be enabled:

- BCMath
- Ctype
- Fileinfo
- JSON
- Mbstring
- OpenSSL
- PDO
- Tokenizer
- XML
- Exif

## Browser Support

InspireCMS admin interface supports:

- Chrome (latest 2 versions)
- Firefox (latest 2 versions)
- Safari (latest 2 versions)
- Edge (latest 2 versions)

## Development Environment

We recommend using one of the following for local development:

- [Laravel Herd](https://herd.laravel.com)
- [Laravel Sail](https://laravel.com/docs/11.x/sail)
- [Laravel Valet](https://laravel.com/docs/11.x/valet)
- [XAMPP](https://www.apachefriends.org/)
- [WAMP](https://www.wampserver.com/en/)
- [MAMP](https://www.mamp.info/en/)

## Server Recommendations

For production environments, we recommend:

- 2GB+ of RAM
- 1+ CPU cores
- 20GB+ of storage space
- A dedicated web server (Apache or Nginx)

## Next Steps

Once you've confirmed your environment meets these requirements, you're ready to [install InspireCMS](./Installing.md).
